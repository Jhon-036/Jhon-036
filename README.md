Soy desarrollador web con experiencia en la creación de aplicaciones interactivas y escalables. Me especializo en React, Redux y JavaScript, y disfruto construir aplicaciones frontend modernas que se conectan de manera eficiente a un backend robusto utilizando Node.js y Express. Además, tengo experiencia trabajando con bases de datos como PostgreSQL y MySql.

* **🔧 Intereses:** Desarrollo de aplicaciones web full-stack con React, Redux, Node.js y Express, y optimización de aplicaciones en la nube usando Azure.
- **🚀 Actualmente aprendiendo:**
  - **Kubernetes** para la gestión de contenedores y orquestación de microservicios.
  - **Azure Active Directory** para gestionar identidades y accesos de manera segura.
  - **Azure Functions** para crear soluciones serverless.
- 🤝 Buscando colaborar en proyectos de open source o en equipos que trabajen con tecnologías modernas y que compartan el interés por la innovación y el desarrollo limpio.
- 💡 Me apasiona explorar nuevas tecnologías y herramientas para mejorar mi flujo de trabajo y mis aplicaciones.
